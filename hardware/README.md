# Hardware Files

This folder contains all hardware-related resources for the Brainy autonomous rover system.

## Purpose
The hardware directory includes all physical design elements of the rover, such as:
- Wiring diagrams  
- Power distribution layouts  
- Connector pinouts  
- Mechanical drawings  
- 3D-printed part files  
- Bill of Materials (BOM)  

## Subfolders (Planned)
- **wiring_diagrams/** – circuits, schematics, connector maps  
- **mechanical/** – SolidWorks files, chassis drawings, mounting plates  
- **bill_of_materials.xlsx** – complete component list with costs  

## Notes
This folder organizes all physical build information needed to assemble, troubleshoot, or upgrade the Brainy rover.
